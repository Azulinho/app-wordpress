app-wordpress Cookbook
======================
Installs wordpress on windoze


Requirements
------------
Needs a windoze box



Usage
-----
#### app-wordpress::default

e.g.
Just include `app-wordpress` in your node's `run_list`:

```json
{
  "name":"my_node",
  "run_list": [
    "recipe[app-wordpress]"
  ]
}
```
